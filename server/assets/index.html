Api end points:

  • Create poll

    POST /poll

    Creates a poll using the parameters specified in the request body

    Returns: The created poll or NULL

  • Get poll

    GET /poll/:pollKey or GET /poll/:pollId

    Finds the poll matching the specified id or key (four word-combination of random words)

    Returns: The found poll or NULL

  • Delete poll

    DELETE /poll/:pollId

    Deletes the poll matching the specified id

    Returns: The deleted poll or NULL

  • Update poll

    POST /poll/:pollId

    Updates the poll matching the specified id using the parameters specified in the request body

    Returns: The updated poll or NULL

  • Get replies

    GET /poll/:pollId/reply

    Gets all replies for the poll matching the specified id

    Returns: An array of replies or an empty array

  • Submit reply

    POST /poll/:pollId/reply

    Creates a reply for the poll matching the specified id using the parameters specified in the request body

    Returns: The created reply or NULL
